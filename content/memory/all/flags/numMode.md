### Address: $D0008A (ti.flags + $0A)

### Size: 1 byte

#### Bit 5 - realMode:
Set for real number mode.

#### Bit 6 - rectMode:
Set for rectangular graph mode.

#### Bit 7- polarMode:
Set for polar graph mode.
