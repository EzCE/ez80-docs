### Address: $D000AB (ti.flags + $2B)

### Size: 1 byte

### Bit 1 - cellOverride:
Set to use cell override.

### Bit 2 - fullScrnDraw:
Set to draw into last row / column of screen. (Not sure if this is applicable to the CE)
