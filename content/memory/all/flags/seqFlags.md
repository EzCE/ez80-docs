#### Bit 0 - webMode:
Set for web mode, reset for normal sequence mode.

#### Bit 1 - webVert:
Seems to be set along with bit 3 for uw mode.

#### Bit 2 - sequv:
Set for uv mode.

#### Bit 3 - seqvw:
Set for vw mode.

#### Bit 4 - sequw:
Set for uw mode. (Might be unused?)

### Address: $D0008F (ti.flags + $0F)

### Size: 1 byte
