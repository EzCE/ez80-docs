### Address: $D00089 (ti.flags + $09)

### Size: 1 byte

#### Bit 1 - parseInput:
Set to parse input when done.

#### Bit 3 - onRunning:
Set if calculator is turned on.

#### Bit 4 - onInterrupt:
Set for on key interrupt request.
