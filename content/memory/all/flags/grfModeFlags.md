#### Bit 4 - grfFuncM:
Set if graph is in function mode.

#### Bit 5 - grfPolarM:
Set if graph is in polar mode.

#### Bit 6 - grfParamM:
Set if graph is in parametric mode.

#### Bit 7 - grfRecurM:
Set for recursion graph.

### Address: $D00082 (ti.flags + $02)

### Size: 1 byte
