#### Bit 2 - curAble:
Set if cursor flash is enabled.

#### Bit 3 - curOn:
Set if the cursor is currently showing.

#### Bit 4 - curLock:
Set if the cursor is locked off.

### Address: $D0008C (ti.flags + $0C)

### Size: 1 byte
