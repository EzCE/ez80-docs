#### Bit 0 - indicRun:
Set if the run indicator is on, reset otherwise.

#### Bit 1 - indicInUse:
Set if the indicator save area is in use, reset if the area is free.

### Address: $D00092 (ti.flags + $12)

### Size: 1 byte
